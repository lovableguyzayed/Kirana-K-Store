Design a complete mobile application UI called "Kirana Konnect", a hyperlocal grocery platform focused on MAP-FIRST discovery of nearby kirana/general stores.

-----------------------------------
CORE IDEA:
This app is MAP-CENTRIC, not search-first.
Users open the app and instantly see a live map with nearby shops, then browse, compare, and order groceries.

-----------------------------------
DESIGN STYLE:
- Modern, clean, minimal UI
- Inspired by Zepto, Blinkit, Google Maps
- Smooth rounded corners (12–16px)
- Soft shadows, layered cards
- High readability

-----------------------------------
COLOR SYSTEM:
- Primary: Green (#2E7D32)
- Secondary: Light Green (#66BB6A)
- Background: White (#FFFFFF)
- Surface: Light Grey (#F5F5F5)
- Text: Dark Grey (#212121)
- Accent: Orange (#FF9800) for CTA highlights

-----------------------------------
TYPOGRAPHY:
- Clean sans-serif (like Inter / Roboto)
- Bold for headings
- Medium for labels
- Regular for body

-----------------------------------
ICON SYSTEM:
- Minimal line icons
- Use grocery-relevant icons:
  - Cart
  - Store
  - Location pin
  - Search
  - Filter
  - Delivery bike
- Consistent stroke width

-----------------------------------
CUSTOMER APP SCREENS:

1. Splash Screen
- Logo centered
- Tagline: "Find nearby kirana stores instantly"

2. Login / OTP
- Mobile input
- OTP verification
- Clean card-based layout

3. MAP HOME SCREEN (PRIMARY SCREEN)
- Full-screen map (Google Maps style)
- User location (blue dot)
- Nearby shop pins (custom grocery icons)
- Floating search bar (top)
- Filter button
- Bottom draggable sheet:
  - Shop cards (name, distance, rating)
- Floating buttons:
  - Recenter location
  - Toggle map/list

4. SHOP PREVIEW (MAP INTERACTION)
- Bottom popup when pin tapped
- Shop name, rating, distance
- "View Shop" CTA button

5. SHOP DETAIL SCREEN
- Shop header (name, rating, distance)
- Category tabs (Dairy, Grocery, Snacks)
- Product list:
  - Product image
  - Name
  - Price
  - Add button
- Sticky cart bar at bottom

6. SEARCH SCREEN (MAP-INTEGRATED)
- Search results show:
  - Shops on map
  - Product list below
- Highlight shops that have searched item

7. CART SCREEN
- Items list with quantity controls
- Remove option
- Toggle:
  - Pickup
  - Delivery
- Price breakdown

8. CHECKOUT SCREEN
- Address selector
- Payment options (COD / UPI)
- Order summary
- Sticky "Place Order" button

9. ORDER TRACKING SCREEN
- Status timeline:
  - Accepted
  - Packed
  - Ready / Out for delivery
- Optional mini map for delivery tracking

10. ORDER HISTORY
- List of past orders
- Reorder button

-----------------------------------
SHOPKEEPER APP SCREENS:

1. Dashboard
- Today's orders
- Earnings
- Pending orders

2. Order Management
- Accept / Reject buttons
- Update status (Packed / Ready)

3. Inventory Management
- Add product form:
  - Name
  - Price
  - Stock
  - Unit (kg, piece, bag)
- Edit / delete options

-----------------------------------
UX PRINCIPLES:

- Map loads instantly
- Shops visible within 1–3 km radius
- Minimal steps to order (max 3–4 taps)
- Smooth transitions (map ↔ list)
- Clear CTAs (Add, View Shop, Place Order)
- Real-time feel

-----------------------------------
OUTPUT REQUIREMENTS:

- High-quality mobile UI mockups
- Consistent spacing and layout
- Realistic grocery products and shop visuals
- Clean alignment and grid system
